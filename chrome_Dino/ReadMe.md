게임 엔진 없이 C# 콘솔 환경에서 입력 처리, 게임 루프, 점프 물리, 장애물 이동, 충돌 판정을 직접 구현한 크롬 공룡 게임 클론 프로젝트입니다.

## 프로젝트 목적

이 프로젝트는 Unity 같은 게임 엔진을 사용하지 않고, C# 콘솔 환경에서 게임의 기본 구조를 직접 구현하는 것을 목표로 합니다.

주요 학습 목표는 다음과 같습니다.

- 게임 루프 구조 이해
- 키보드 입력 처리 구현
- 점프와 중력 기반 이동 구현
- 장애물 이동 및 재생성 구현
- 좌표 기반 충돌 판정 구현
- 역할별 클래스 분리 연습
